# KohliBot

An AI-powered cricket coaching web app built with Flask and OpenRouter API.

## Features
- Video Analysis — Upload batting video, get AI feedback
- Batting Coach — Personalized coaching in Kohli's voice
- Communication Skills — Body language and confidence
- Leadership & Mindset — Captaincy philosophy

## Tech Stack
- Python, Flask
- OpenRouter API (AI)
- HTML, CSS, JavaScript

## Setup
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your OpenRouter API key to `.env`
4. Run: `python app.py`
